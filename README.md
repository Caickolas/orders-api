# Orders API

API desenvolvida em **Node.js** com **Express** e **MongoDB** para gerenciamento de pedidos.
A aplicação permite criar, consultar, listar, atualizar e remover pedidos através de endpoints HTTP.

---

# Tecnologias utilizadas

* Node.js
* Express
* MongoDB
* Mongoose
* JavaScript

---

# Como executar o projeto

### 1. Clonar o repositório

```
git clone https://github.com/seu-usuario/orders-api.git
```

### 2. Entrar na pasta do projeto

```
cd orders-api
```

### 3. Instalar dependências

```
npm install
```

### 4. Iniciar o MongoDB

Certifique-se de que o MongoDB está rodando localmente.

### 5. Executar a aplicação

```
node server.js
```

Servidor iniciará em:

```
http://localhost:3000
```
